
# 🛏️ Madhumitha Tex

A modern, responsive textile e-commerce website built for **Madhumitha Tex**, showcasing premium home textile products with an intuitive shopping experience. The platform allows customers to browse products, view detailed information, and place orders directly through WhatsApp. An integrated admin panel enables seamless product management.

---

## 🌐 Live Website

🔗 https://www.madhumithatex.co.in

### Customer Features

- 🛍️ Browse premium textile collections

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/your-username/madhumitha-tex.git
```

Navigate to the project

```bash
cd madhumitha-tex
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```

---

## ⚙️ Environment Variables

Create a `.env` file in the root directory.

```env
VITE_SUPABASE_URL=YOUR_SUPABASE_URL
VITE_SUPABASE_ANON_KEY=YOUR_SUPABASE_ANON_KEY
```
